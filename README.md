# React + TypeScript + Vite + Material Ui

## To run the app 
`` git clone https://github.com/LouisKzy/Mui-test  ``

Then install nessaray packages with your package manager

`` yarn install``
or
`` npm install``

## Uncomment the components you want to see
Then run 
``yarn start``
or
``npm run dev``
